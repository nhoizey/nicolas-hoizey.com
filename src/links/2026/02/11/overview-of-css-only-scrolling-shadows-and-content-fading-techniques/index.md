> When content overflows and the scrollbar is insufficient or intentionally hidden, visual cues such as scroll shadows or fading effects signal that additional material extends beyond the visible region.
> 
> This is an overview of five CSS techniques for implementing these cues, for both vertical and horizontal scrolling.
